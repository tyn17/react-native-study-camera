# react-native-study-camera

The package to open Camera for Belle Study application

## Installation

```sh
npm install react-native-study-camera
```

## Usage

```js
import { StudyCameraView } from "react-native-study-camera";

// ...

<StudyCameraView color="tomato" />
```
